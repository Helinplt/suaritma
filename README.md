💧 Su Arıtma Web Sitesi Tasarımı

Bu proje, su arıtma sistemleri tanıtımı ve ürün listeleme amacıyla geliştirilmiş statik web sitesi tasarım projesidir.  
HTML, CSS ve JavaScript kullanılarak hazırlanmıştır ve GitHub Pages üzerinde yayınlanmaya uygundur.

Proje; ev tipi arıtma, sanayi tipi arıtma ve yedek parça ürünlerini listeleyen, detay sayfaları bulunan modern bir web arayüzü içerir.

---

## 🔧 Proje Hakkında

Bu web sitesi;

- Kurumsal su arıtma firması için hazırlanmış örnek tasarım
- Ürün listeleme ve detay sayfası sistemi
- Responsive (mobil uyumlu) yapı
- GitHub Pages uyumlu statik site

olarak geliştirilmiştir.

Site içerisinde:

- Anasayfa
- Ev Tipi Arıtma Sayfası
- Sanayi Tipi Arıtma Sayfası
- Yedek Parça Sayfası
- Ürün Detay Sayfaları
- Galeri
- İletişim

bölümleri bulunmaktadır.
